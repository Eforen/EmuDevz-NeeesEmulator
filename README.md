# NEEES emulator (EmuDevz)

This repository contains my code for a **NEEES** emulator written while playing **EmuDevz**, a game about building emulators.

- [EmuDevz on itch.io](https://afska.itch.io/emudevz)
- [EmuDevz source / upstream on GitHub](https://github.com/afska/emudevz)

## Spoilers

Working through EmuDevz is meant to be part of the experience. **This repo may spoil puzzles, progression, or implementation details** from the game. If you care about that, play first and come back later.
